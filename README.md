# Sistema de Cadastro de Alunos

## Descrição
Projeto desenvolvido para a disciplina de Algoritmos e Programação (A3).

O sistema permite o gerenciamento de alunos através de um menu interativo no terminal.

## Funcionalidades

- Cadastrar aluno
- Listar alunos
- Editar aluno
- Excluir aluno
- Encerrar sistema

## Estrutura do Projeto

### Main.java
Responsável pelo menu e interação com o usuário.

### Aluno.java
Classe que representa a entidade Aluno.

### AlunoService.java
Responsável por gerenciar os alunos cadastrados.

## Tecnologias Utilizadas

- Java
- VS Code
- GitHub

## Como Executar

1. Compilar os arquivos Java
2. Executar a classe Main
